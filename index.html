New here?

Modelizer Next helps you plan a database before you build it.

Instead of jumping straight into SQL, you draw the structure first. Add tables, add fields, connect related data, and move between conceptual, logical, and physical views as your idea gets more precise.

Get startedSee controls
1. Create or open a model

Start a new file or import an existing .mn, .mod, or .mdlz model.

2. Add tables and fields

Use the toolbar or shortcuts to build the database shape piece by piece.

3. Link related data

Create links between tables to show how records connect.

4. Save and continue later

Keep your model as a file, share it, or use it as a base for implementation.

Built for existing Modelizer users

Move over without changing the way you model.

Modelizer Next is made as a direct replacement for the classic Modelizer tools such as Modzlier or modelizer.lu. It supports existing model formats, keeps the core database modeling ideas, and removes the old, clunky UI feeling.

Controls, shortcuts, and dialogs

Fast actions without hunting through menus.

The app keeps common actions close to the keyboard and mouse. Use the toolbar when you want buttons, or shortcuts when you want speed.

Keyboard shortcuts

CtrlNNew document
CtrlOLoad a model file
CtrlSSave
CtrlShiftSSave as
CtrlZUndo
CtrlShiftZRedo
CtrlTAdd table
CtrlFAdd field
ShiftCAdd comment
CtrlLAdd link
F2Rename selected item
CtrlEEdit selected item
DeleteDelete selected item
CtrlDDuplicate selected item
EscClear selection
CtrlASelect all
CtrlCCopy selection
CtrlXCut selection
CtrlVPaste selection
↑↓Move through fields
Alt↑Alt↓Move a selected field in the list

Mouse controls

Left drag moves objects. Middle drag pans the canvas. The mouse wheel zooms. Right drag creates a link between any objects.

File dialogs

Open, save, save as, and file drop dialogs keep model files simple.
Supported files: .mn, .mod, and .mdlz.

Appearance

Switch between light, dark, and system theme modes. Manage style palettes from the Styles menu.
