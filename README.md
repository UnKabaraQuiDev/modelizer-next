### Features:
<table cellspacing="10">
  <tr>
    <td>
      <strong>Load existing files</strong><br>
      Supports <code>.mod</code>, <code>.mdlz</code> and <code>.mn</code>
    </td>
  </tr>
  <tr>
    <td>
      <strong>Export models as images</strong><br>
      Supports <code>.svg</code> and  <code>.png</code>. A command line export option is also present, try `modelizer-modelizer -e --help` for information
    </td>
  </tr>
  <tr>
    <td>
      <strong>Three views model</strong><br>
      Design split into 3 views: conceptual, logical and physical
    </td>
  </tr>
  <tr>
    <td>
      <strong>Visual customization</strong><br>
      Add you own styles to color tables, links and comments
    </td>
  </tr>
  <tr>
    <td>
      <strong>Dark/Light Themes</strong><br>
      Supports dark/light/follow system themes with a clean interface
    </td>
  </tr>
  <tr>
    <td>
      <strong><a href="https://modelizer.kbra.lu/#shortcuts" target="_BLANK">Intuitive controls</a></strong><br>
      Lots of keyboard shortcuts and mouse gestures
    </td>
  </tr>
</table>

### Build types:
<table align="center" cellspacing="10">
  <tr>
    <td>
      <strong>Update installer</strong><br>
      Installs an updater that allows you to choose your update channel (nightly, snapshot or release). Allows for automatic updates on startup. Recommended for most use cases.
    </td>
  </tr>
  <tr>
    <td>
      <strong>Standalone installer</strong><br>
      Installs a fixed version without automatic updates. Useful for environments where updates are not needed or no permanent connection is available.
    </td>
  </tr>
  <tr>
    <td>
      <strong>Portable</strong><br>
      Portable zip, no installation required. Useful for USB sticks or automated distribution on different machines.
    </td>
  </tr>
</table>
